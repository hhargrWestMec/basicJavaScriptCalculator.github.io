Calculator

This calculator can be used for small numbers only. It will round to the tenths power each time. There is no clear button because the 'Backspace' key has been used to clear the numbers. One click will remove the current number, and if you wish to reset the whole history, click twice. After each use, make sure to clear the history by hitting the 'Backspace' key on your keyboard. The number keys and operator keys can be used, but the 'Enter' key will not submit the answer. The 'Equal' key can be used to submit your answer. When the "TOO LARGE" message appears, you may continue by typing a new number. The 'Backspace' key will not clear this message.
